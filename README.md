# DevOps Pipeline WebApp

Java WebApp mit CI/CD Pipeline auf Raspberry Pi 5.

## Features
- Java 21 + Servlets/JSP
- MySQL 8.0
- Docker Compose (Jenkins, Tomcat 11, MySQL)
- Automatisches Build & Deployment

## Zugriff
- WebApp: http://localhost:8080/webapp/
- Jenkins: http://localhost:8081/

## Pipeline Flow
Push → GitHub → Jenkins → Build → Test → Deploy → Tomcat
